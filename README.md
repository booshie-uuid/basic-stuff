# MY (Q)BASIC STUFF

A repository to preserve my old QuickBASIC projects as I recover and test them.

* **AMAZE.BAS**: A cool little random maze generator.
* **QB3D.BAS**: Rotates and draws basic 3D shapes (cube, pyramid, and "sphere").

Honestly there is nothing special about this code. There are much cleaner and cleverer examples out there.

Preserved for nostolgia and curiosity only.

I have attempted to port these programs to a modern transpiler like QBasic Nexus, but I haven't had much luck. It too often required changing the code more than I was comfortable doing.

I have tested all these programs still works well in QBasic 1.1 on FreeDos (via DosBox) before uploading them.
